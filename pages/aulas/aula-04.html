Definição de BootStrap

Bootstrap é um framework front-end que fornece estruturas de CSS para a criação de sites e aplicações responsivas de forma rápida e simples. Além disso, pode lidar com sites de desktop e páginas de dispositivos móveis da mesma forma.

Versões do Bootstrap

BootStrap 3

  • é baseado em um sistema de grade de 4 camadas (xs, sm, md, lg).
  • no bootstrap 3, a classe .responsive-table adicionada ao elemento < div>.
  • cores limitadas estão disponíveis, ele suporta barras de navegação inversas, mas não outras classes.
  • a classe .row não é necessária usando grade em formulários.

BootStrap 4

  • é baseado em um sistema de grade de 5 camadas (xs, sm, md, lg, xl).
  • no bootstrap 4, a classe .responsive-table adicionada usando o elemento < table> .
  • existem muitas opções de cores .bg-dass ou .navbar-light, .navbar-dark classes
  • a classe .row é necessária ao usar a grade no formulário.

BootStrap 5

  • uma aparência diferente em diferentes sistemas operacionais e navegadores. A aparência dos elementos do formulário não mudará em sistemas operacionais ou navegadores diferentes.
  • Bootstrap 5 deu liberdade para modificar e também criar nosso próprio utilitário
  • A propriedade Inline-block foi removida e teremos um menu suspenso preto como padrão para a classe dropdown-menu-dark.

Implementando BootStrap

Para o desenvolvimento de templates com Bootstrap externo, é necessário a sua vinculação pelo link CSS. Para aplica-lo de forma offline devemos baixar todo o seu conteudo e colormos em um documento CSS, declarando seu endereço de referência. Tanto seu sdocumento CSS como JS deve ser aplicados da mesma forma
